# Prompt Examples

These prompts can be copied directly into Codex for use.

## Planning illustrations only

```text
Use $ian-xiaohei-illustrations. Don't generate images yet.
Analyze the article below for where illustrations would add value, and output a shot list of about 5 images.
For each image, specify:
- Which paragraph it follows
- Image theme
- Core meaning
- Structure type
- What Xiaohei is doing in the image
- Suggested elements
- Suggested English annotation words

<paste article>
```

## In-article illustrations

```text
Use $ian-xiaohei-illustrations to generate 4 Xiaohei absurdist in-article illustrations for the article below.
Requirements: 16:9 landscape, pure white background, black hand-drawn line art, sparse red/orange/blue handwritten English annotations.
Each image should convey only one core structure — no PPT infographics, no cute cartoons.

<paste article>
```

## Illustration strategy for long articles

```text
Use $ian-xiaohei-illustrations to develop an illustration strategy for this long article.
Don't distribute illustrations evenly — pick only cognitive anchors: core judgments, input/output loops, before/after comparisons, common pitfalls, handoff paths.
Default to 6-8 images, output the shot list first, don't generate images yet.

<paste article>
```

## Generating one image for a single idea

```text
Use $ian-xiaohei-illustrations to generate one 16:9 in-article illustration for this idea:

Trust isn't shouted into existence — it's laid down piece by piece, one piece of evidence at a time.

The image should be quirky but clean, and Xiaohei must carry out the core action.
Use at most 5 English annotations, keep them short.
```

## Workflow theme

```text
Use $ian-xiaohei-illustrations to generate an image for "processing one piece of raw material into three types of content: traffic, trust, and conversion."
Don't draw a formal flowchart, and don't replicate the old "One Fish, Many Uses" example.
Reinvent a new low-tech metaphor, and have Xiaohei participate in the core action.
```

## Editing an image: remove the title

```text
Use $ian-xiaohei-illustrations to help me edit this image.
Remove the "Workflow" title and underline in the top-left corner, keep everything else unchanged.
Don't add any new text or objects.
```

## Editing an image: strengthen Xiaohei's involvement

```text
Use $ian-xiaohei-illustrations. This image is on the right track, but Xiaohei feels a bit decorative.
Keep the core meaning unchanged and regenerate a version where Xiaohei is the one truly driving the structure.
Make the image a bit weirder, but still pure white, clean, and with minimal text.
```

## Generating a batch of style samples

```text
Use $ian-xiaohei-illustrations to output 5 Xiaohei in-article illustrations on different themes.
Themes: information overload, product validation, content compounding, one-person company, building trust.
Generate each one separately, don't combine them into a single image.
```
